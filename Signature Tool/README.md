# SignaturePro | Professional PDF Signing Tool

SignaturePro is a premium, web-based tool designed for seamless PDF document signing. It offers a professional user interface with advanced features for creating, managing, and placing signatures with precision.

## ✨ Features

- 📑 **PDF Upload & Viewing**: High-fidelity PDF rendering with page navigation.
- ✍️ **Custom Signature Pad**: Draw your signature with adjustable pen thickness and colors.
- 🏦 **Signature Bank**: Store multiple signatures and reuse them across different documents.
- 🎯 **Interactive Editor**: Drag-and-drop signatures directly onto the PDF.
- 🔍 **Zoom & Pan**: Precise control with zoom functionality for accurate placement.
- 💾 **Instant Download**: Generate and download your signed PDF in seconds.

## 🚀 Technologies Used

- **HTML5 & Vanilla CSS**: For a modern, responsive, and premium UI.
- **JavaScript (Vanilla)**: For core application logic.
- **[PDF-lib](https://pdf-lib.js.org/)**: For PDF modification and generation.
- **[PDF.js](https://mozilla.github.io/pdf.js/)**: For high-quality PDF rendering.
- **[Signature_Pad](https://github.com/szimek/signature_pad)**: For smooth signature drawing.
- **[Interact.js](https://interactjs.io/)**: For drag-and-drop and resizing functionality.
- **[Lucide Icons](https://lucide.dev/)**: For beautiful, consistent iconography.

## 🛠️ Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/signature-tool.git
   ```
2. **Open the project**:
   Simply open `signatureTool.html` in any modern web browser. No server setup required!

## 📖 How to Use

1. **Upload**: Select your PDF file from the "Upload PDF" tab.
2. **Create**: Go to the "Create Signature" tab, draw your signature, and click "Add to Bank".
3. **Edit**: Switch to the "Editor" tab. Drag your signature from the sidebar onto the PDF.
4. **Position**: Move and scale the signature as needed.
5. **Download**: Click "Download PDF" to save your signed document.
